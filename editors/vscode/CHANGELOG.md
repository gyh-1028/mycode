# Changelog

## 0.2.2

- Synchronize extension version with MyCode `0.2.2`.
- `run/result` notification now includes optional `tracePath` so the UI can
  link to local trace files.

## 0.2.1

- Synchronize extension version with MyCode `0.2.1`.
- No user-facing changes; protocol v1 schema documents optional tool-event
  payload fields (`tool_call_id`, `duration_ms`).

## 0.2.0

- Initial VS Code workspace extension.
- Session tree, agent conversation, tool activity, cancellation, permissions,
  editor selection context, and JSON-RPC protocol v1 support.
